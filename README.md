# Expense Tracker (Python)

A simple command-line based expense tracker built using Python.

## 🚀 Features

- Add expense (amount + category + note)
- View all expenses
- Calculate total expense
- File-based storage using JSON

## 📥 Input

- Amount
- Category
- Note

## 📤 Output

- Expense list
- Total expense summary

## ▶️ How to Run

`bash
python main.py
